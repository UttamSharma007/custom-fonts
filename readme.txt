== Description ==

Bsf Custom Fonts allows you to add as many custom fonts to extends the font support in your theme.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bsf-custom-fonts` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->BSF Custom Fonts Add Custom Fonts name, woff2, woff, ttf, eot and svg files.
